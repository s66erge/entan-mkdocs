# MAIN STRUCTURES



## Main folders and files

- `\data\`:
    -   gong center databases (.db): planning and timetables
    -   gong centers configurations (.xlsx): see the app STATUS and CONFIGURATION page 
- `\docs\`:
    -   THIS documentation of architecture, system and dev/prod environments for the gong center app and web app. It contains and is automatically synchronized with the web app source code (see Entangled in `Setup deploy`)
- `\libs\*.py` + `\main.py`: source code for the gong web app
- `\md-text\`: user documentation integrated in the web app
- `\setup\`: scripts to setup the gong web app and its dev tools (see `Setup deploy`)
- `\site\`: static website files generated by mkdocs from the `\docs` (sub)folder(s)
- `\tests\`: unit tests for the gong web app
- 
- '\pyproject.toml and uv.lock': configuration used by `uv` for the gong web app dependancies
- `\*.yaml and .toml`: configuration files for the gong web dev tools (see `Setup deploy`)
- `\Dockerfile`: Dockerfile to build the gong web app production container
