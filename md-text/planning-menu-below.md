- **(re)Start getting plans**: (re)loads the current center gong plan and the dhamma.org current plan, combines them and shows if there are issues for the gong planning. If you loaded it already and you made changes, they will be lost.
- **Load saved plan**: shows the plan with the changes you have already made since the last '(re)Start getting plans'. Allows you to go work on the timetables and then return working on the plan where you left it. 
- **(re)Start timetables**: (re)loads the current center gong timetables and checks that it fits the gong planning constraints. If you loaded them already and you made changes, they will be lost.
- **Load saved timetables**: shows the timetables with the changes you have already made since the last '(re)Start getting plans'. Allows you to go work on the plan and then return working on the timetables where you left them.
- **return NO CHANGES** and **SAVE ALL CHANGES**: see above help text **How to save your work**
- **Download PDF**: downloads a pdf copy of this entire page. Remember to first close down help text(s) you do not want to print.
PDF FORMATTING suggestion: landscape, max. 80% scale, minimum margins and background graphics
