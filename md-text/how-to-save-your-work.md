While you work on the plan and timetables, your work is temporarily saved IF YOU REMAIN ON THIS PAGE. You must leave this page by clicking either:
  - the button 'SAVE ALL CHANGES', and then confirm with 'OK' and then 'Leave'. This process is irreversible and no user will be able to do any change in the meantime. 
  You can then leave this page with the new link 'return to STATUS after SAVING ALL CHANGES'.
  - the link 'return NO CHANGES' to abandon changes and return to the Dashboard

Clicking on a link on the second row will open a page in a new browser tab where you can consult info about this or other centers.
**Then you must return to this browser tab and leave this page** ONLY with 'SAVE ALL CHANGES' (followed by 'return to STATUS after SAVING ALL CHANGES') or 'return NO CHANGES'. Failure to do so will definitively cancel any change since the start of your session.

As nobody else can change this center plan/timetables as long as you are on this page, there is a countdown timer which will automatically trigger 'return NO CHANGES' after one hour.
